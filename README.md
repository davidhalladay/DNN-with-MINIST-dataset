# DNN-with-MINIST-datasetx
## Introduction
Apply the DNN method on the MINIST data set.

## brief report
Accuracy : above 99.8%

## Setting
1. install the DNN tool tensorflow and a high level api Keras
2. python version >= 3.5, 64-bit
3. run the MINIST.py in terminal
